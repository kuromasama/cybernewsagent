# 🚨 雲端惡意軟體框架VoidLink：AI生成的複雜攻擊工具
> **⚡ 戰情快篩 (TL;DR)**
> * **嚴重等級**: Critical (CVSS: 9.8)
> * **受駭指標**: RCE (Remote Code Execution)
> * **關鍵技術**: `eBPF`, `Serverside Rootkit Compilation`, `AI生成惡意軟體`

## 1. 🔬 漏洞原理與技術細節 (Deep Dive)
* **Root Cause**: VoidLink採用了「伺服器端編譯Rootkit」機制，允許攻擊者根據受害主機的核心版本與設定，即時編譯相容的核心Rootkit。
* **攻擊流程圖解**:
  1. 攻擊者要求AI根據規格取向的開發方法（Spec-Driven Development，SDD）生成惡意軟體。
  2. AI生成的惡意軟體包含了「伺服器端編譯Rootkit」機制。
  3. 攻擊者使用C2伺服器控制受害主機，根據受害主機的核心版本與設定，即時編譯相容的核心Rootkit。
* **受影響元件**: Linux核心版本（未指定特定版本）

## 4. 📚 專有名詞與技術概念解析 (Technical Glossary)
* **eBPF (extended Berkeley Packet Filter)**: 一種Linux核心技術，允許用戶空間程式碼在核心空間執行，常用於網路監控和安全分析。
* **Serverside Rootkit Compilation**: 一種攻擊技術，允許攻擊者根據受害主機的核心版本與設定，即時編譯相容的核心Rootkit。
* **AI生成惡意軟體**: 一種攻擊技術，使用人工智慧算法生成惡意軟體，常用於自動化攻擊和繞過防護機制。
