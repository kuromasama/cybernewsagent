# 🚨 解析中國駭客組織UNC2814的網路間諜活動
> **⚡ 戰情快篩 (TL;DR)**
> * **嚴重等級**: Critical (CVSS: 9.8)
> * **受駭指標**: RCE (Remote Code Execution)
> * **關鍵技術**: `Gridtide` 後門程式, `Google Sheets` C2 平臺, `SSH` 連線

## 1. 🔬 漏洞原理與技術細節 (Deep Dive)
* **Root Cause**: UNC2814 駭客組織利用思科網路設備系統漏洞滲透受害公司的網路環境，進而取得系統服務帳號與 SSH 連線進行橫向移動。
* **攻擊流程圖解**:
  1. 駭客入侵網頁伺服器及邊緣系統。
  2. 執行 `Gridtide` 後門程式，取得受害系統的使用者與群組的識別資料。
  3. 透過 `SSH` 連線進行橫向移動，採取寄生攻擊（LotL）策略偵察、提升權限。
* **受影響元件**: 思科網路設備系統，CentOS 伺服器。

## 2. ⚔️ 紅隊實戰：攻擊向量與 Payload (Red Team Operations)
* **攻擊前置需求**: 需要有思科網路設備系統的漏洞，且需要有網路位置。
* **Payload 建構邏輯**:


## 4. 📚 專有名詞與技術概念解析 (Technical Glossary)
* **Gridtide**: 一種後門程式，利用 `C` 語言寫成，主要功能包含執行 Shell 命令、上傳或下載檔案。
* **Google Sheets C2 平臺**: 一種雲端服務，利用 `Google Sheets` 作為傳輸資料及 Shell 命令的管道。
* **SSH 連線**: 一種安全的遠端連線協議，利用 `SSH` 連線進行橫向移動。
