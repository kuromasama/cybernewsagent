# 🔥 RSA 金鑰中質因數的特殊分布：解析和利用
> **⚡ 戰情快篩 (TL;DR)**
> * **嚴重等級**: High (CVSS: 7.5)
> * **受駭指標**: Factorization of RSA keys
> * **關鍵技術**: RSA, Factorization, Polynomial Methods

## 1. 🔬 漏洞原理與技術細節 (Deep Dive)
* **Root Cause**: RSA 金鑰中的質因數具有大量連續零位元（zero bits）等特殊分布，導致這些金鑰更容易被破解。
* **攻擊流程圖解**: 
    1. 收集大量實際使用的 RSA 金鑰
    2. 分析金鑰中的質因數
    3. 發現具有特殊分布的質因數（例如大量連續零位元）
    4. 利用多項式方法降低這類 RSA 金鑰質因數分解的難度
* **受影響元件**: RSA 金鑰，尤其是那些具有特殊分布的質因數的金鑰

## 4. 📚 專有名詞與技術概念解析 (Technical Glossary)
* **RSA (Rivest-Shamir-Adleman)**: 一種公鑰加密算法，使用大質數的乘積作為金鑰。
* **Factorization (分解)**: 將一個數字分解為其質因數的過程。
* **Polynomial Methods (多項式方法)**: 一種數學方法，使用多項式方程來解決問題。
